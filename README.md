# FCC Course: Data Analysis with Python

## Setup the Virtual Environment:

- `cd myprojectDir`
- create Venv (.myvenv)
- activate .myvenv
- `pip install numpy`
- `pip install matplotlib`
- `pip install pandas`

## Launch Jupyter Lab and use Venv

- `cd myProjectDir`
- `jupyter lab`
- in Jupyter Lab, kernel -\> change kernel -\> select ".myenv"
